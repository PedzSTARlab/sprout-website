SPROUT Metadata

This section outlines the standardized BIDS-inspired folder structure used in the SPROUT dataset. It includes raw audio, associated metadata, and multiple stages of audio processing pipelines (denoising, normalization, QC).

Dataset File Structure Overview

> PedzSTAR_PublicNonCommercial_Dataset/  
> ├── dataset_description.json

Task-specific files follow the naming convention:  
sub-[ParticipantID]_task-[TaskName]_acq-[AcquisitionType][_run-<index>]_desc-rawdata.wav / .json  
Processed files follow:  
sub-[ParticipantID]_task-[TaskName]_acq-[AcquisitionType][_run-<index>]_desc-denoisednorm.wav / .json, etc.
