Task Reference Descriptions

Task Task Code Description
Toy Play TPL Children engage in spontaneous play with a farm-themed toy set to encourage natural speech and interaction.
Picture Task PCT Children complete two activities: (1) answer open-ended questions, and (2) repeat specific words, both while viewing a series of images.
GFTA (Goldman-Fristoe Test of Articulation) GFTA Children name pictures using single words that target a variety of speech sounds for articulation assessment.
PLS-5 (Preschool Language Scale â€“ Fifth Edition) PLSS Children respond to age-appropriate questions and label pictures to assess vocabulary, language skills, and speech sound production.
CELF-P3 Screener CELFS Children complete sentences aloud using correct grammar while viewing pictures, focusing on early syntax and concept understanding.
CELF-P3 Tester CELFT Children perform two tasks: (1) complete sentences using correct grammar and concepts, and (2) name pictures to assess expressive vocabulary.
