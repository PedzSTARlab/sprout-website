Data Governance

The SPROUT project implements robust data governance practices to ensure ethical use of child speech data.

Ethical Protections

  • All data collection follows rigorous human subjects research protocols
  • Informed parental consent with clear explanation of data usage
  • Options for data sharing preferences
  • Strict data privacy and security measures

Data Access

Access to the SPROUT dataset is managed through a controlled access mechanism:

  • Data Use Agreement (DUA) required
  • Data Access Request Form submission
  • Review by Data Access Committee
  • Institutional signature requirements

Data Security

  • Secure storage with encryption
  • Access controls and authentication
  • Audit logging of data access
  • Regular security assessments

FAIR Principles

The SPROUT dataset adheres to FAIR data principles:

  • Findable: Persistent identifiers and rich metadata
  • Accessible: Clear access protocols while protecting privacy
  • Interoperable: Standard formats and vocabularies
  • Reusable: Clear licenses and provenance information

Detailed data governance documentation and access request forms will be added in a future update.
