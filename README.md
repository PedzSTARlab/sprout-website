## Overview

The SPROUT (Speech Production Repository for Optimizing Use for AI Technologies) project is designed to support the development of automatic speech recognition (ASR) systems and AI-driven tools for early childhood assessment. SPROUT focuses on collecting speech samples from four-year-old children‚Äîa strategic choice that targets the developmental midpoint of speech sound acquisition in American English.
